# MVC
    - M:model データとデータに係るロジック（アプリケーション特有の処理）をオブエジェクトとして実装。DBとのやり取り
    - V:view UI。HTMLなどのHTTPレスポンスの中身を組み立てる
    - C:Controller モデルやビューを統合的に制御する

# コントローラ層がリクエストを受ける係
    ある機能を加えるために、まずその機能をどのコントローラ（クラス）のなんのアクション（メソッド）で処理するかを決める
